# Java
* System.out对象下的println方法
* 回车不是语句结束的标志，可以把一个语句分开多行写
* 长整型常数后面加L，单精度浮点型加F
* \u转义序列是在代码解析之前得到直接替换的处理，参见java核心技术P34下方的警告
* Java中的数值不能作真值
* Java中的数组声明需要用new
* private修饰的成员是所有其他类都无法访问的，即使是经过继承的子类也无法访问。虽然子类可以通过构造器修改private实例域的值，但是无法真正的访问到private实例域
* *VSCode中只要有一个java文件有错误就无法进行build*
* 抛出异常的主体是方法而不是类
* 当一个方法return一个对象时，并没有像基本数据类型那样创建一个副本对象，而是把对象的引用传回去了。接收这个对象的变量引用的对象和方法中使用的对象是同一个对象
