# 函数
### 函数的调用机制(概要)
在一个函数执行过程中如果需要**调用**其他函数，一般会经过以下的过程
1. 保留当前函数现场
2. 调用函数
3. 恢复当前函数现场
4. 继续执行当前函数
### 函数的调用机制  
1. 建立栈帧空间
2. 主调函数运行状态和返回地址入栈    *这里对应的是“保留函数现场”*  
3. 为形参建立存储空间并同时传入数据，实现形参和实参的结合，接着给局部变量分配空间
4. 执行函数体
5. 执行完成后释放局部变量占用的栈空间
6. 主调函数主调函数运行状态和返回地址出栈      *这里对应的是“恢复函数现场”*  
7. 继续执行主调函数

从这个过程可以看出函数调用的开销在于**保存和恢复现场**即状态和返回地址出入栈的这个过程
